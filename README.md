# Nantian Gateway Dashboard

Next.js 16 + React 19 admin console for the Nantian Gateway.

## Features

- **Gateway Management** — Configure and monitor gateway routes, backends, and policies
- **Observability** — Real-time metrics, logs, and health monitoring
- **AI Chatbot** — Integrated AI assistant for gateway operations

## Tech Stack

| Component    | Technology                          |
|-------------|-------------------------------------|
| Framework    | Next.js 16 (App Router)            |
| UI Library   | React 19                           |
| Styling      | Tailwind CSS v4 + shadcn/ui        |
| Auth         | NextAuth.js                        |
| Language     | TypeScript                         |

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The dev server runs at `http://localhost:3000`.

## Lint & Type Checking

```bash
npm run check
```
